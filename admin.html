Cách cập nhật trang web sau khi chỉnh sửa

  1. Nhấn "Tải xuống data.json" bên dưới
  2. Thay thế file data.json cũ trong thư mục dự án
  3. Chạy lệnh: git add data.json && git commit -m "update" && git push
  4. GitHub Pages tự động cập nhật trong vài phút ✓

🚀 Hướng dẫn deploy lên GitHub Pages (miễn phí)

  1. Tạo tài khoản GitHub tại github.com nếu chưa có
  2. Tạo repository mới, đặt tên bất kỳ (VD: travel-blog)
  3. Upload 3 file: index.html, admin.html, data.json lên repo
  4. Vào Settings → Pages, chọn Source: Deploy from a branch → main → / (root)
  5. Nhấn Save. Đợi ~1 phút, trang web sẽ live tại https://username.github.io/travel-blog

📸 Cách lấy URL ảnh từ Google Photos

  1. Mở photos.google.com
  2. Click vào ảnh muốn dùng
  3. Click nút Chia sẻ (biểu tượng mũi tên)
  4. Chọn "Tạo liên kết"
  5. Copy URL — dán vào ô URL ảnh trong admin

⚠ Lưu ý: Google Photos đôi khi giới hạn ảnh public. Nếu ảnh không hiển thị, hãy dùng Google Drive: upload ảnh → share "Anyone with link" → lấy file ID → dùng URL dạng https://drive.google.com/uc?id=FILE_ID

💡 Workflow hàng ngày

  • Mở admin.html trực tiếp trong trình duyệt (không cần server)
  • Thêm/sửa chuyến đi → Xuất → Tải data.json → Push lên GitHub
  • Trang public tự cập nhật, không mất phí gì cả ✓
